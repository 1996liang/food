# food
作为我的一个项目，是一个较为完善的B2C商城，商品标签，商品选项，购物车，订单管理,地址管理，个人中心，评价，商品列表，商品搜索等等功能都有<br>
同时它有后台管理系统，支持商品上下架，管理用户订单，增添商品分类等等功能<br>
